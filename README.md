# (C# 코딩) 에코메신저

## 개요-C# 프로그래밍학습
-핵심기능: 실시간 메세지 전송, 문자열 가공, 키보드 인터랙션

-화면구성  
-Form: 프로그램 전체의 윈도우 창으로, 타이틀바(Text 속성)에 "Echo Messenger"라고 표시합니다.  
-Label: 화면에 "에코 메신저" 또는 "Echo Messenger"와 같은 안내 문자열을 표시합니다.  
-TextBox: 사용자가 메시지를 타이핑하여 입력할 수 있는 공간입니다.  
-Button: 입력한 메시지를 전송하거나, 특정 항목을 삭제하는 등의 동작을 수행합니다.  
-ListBox: 전송된 메시지 이력을 리스트 형태로 보여주는 대화창 역할을 합니다.
